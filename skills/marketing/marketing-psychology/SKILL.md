Apply persuasion psychology to marketing so a true offer reaches the right decision faster, never to engineer a choice the prospect would regret. The discipline is choosing the principle that fits the action, applying it on a claim that is literally true, and stopping at the line where persuasion turns into manipulation. The full catalogue of principles, biases, the ethics tests, the dark-pattern list, the failure modes, and a worked pricing-page example live in [references/persuasion-principles.md](references/persuasion-principles.md).

Read that reference before applying any principle below.

## Steps

1. **Name the desired action.** State the one specific thing the prospect should do on this asset — a Pro signup, an email subscribe, a demo booking — and who the qualified prospect is. The step is done when a single named action and audience are written down, not a vague "more conversions".

2. **Pick the fitting principle(s).** Match the named action to one or two principles from the reference (Cialdini's seven, or a conversion bias) that suit this audience and decision. The step is done when each chosen principle is paired with the specific reason it fits this action, and no more than two are selected.

3. **Apply each principle honestly.** Write the copy, offer, or layout element that expresses each chosen principle. The step is done when every applied principle has a concrete on-page element and a one-line note of the truth condition it depends on, drawn from the reference.

4. **Check against the dark-pattern red flags.** Test the draft against the forbidden list in the reference — fake scarcity, forced continuity, confirm-shaming, hidden costs, bait-and-switch. The step is done when each red flag is marked absent, or the offending element is rewritten until absent.

5. **Verify every claim is true.** Confirm each number, deadline, testimonial, badge, and limit against a real source of truth. The step is done when each claim links to its evidence, and any claim without evidence is cut from the asset.

6. **Run the three ethics tests.** Apply the transparency, truth, and interest tests from the reference to the whole asset, and reject a tactic that fails even one. The step is done when a written verdict records a pass on the three tests against the one or two principles chosen in step 2.

7. **Define the measurement.** State the primary conversion metric for the named action plus one downstream guardrail (90-day retention or refund rate) that catches trust damage. The step is done when both metrics are named and a single-variable A/B test is specified to isolate each principle's effect.
