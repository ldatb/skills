Structure a marketing website the way it must earn its keep: move a stranger from "never heard of you" to "took the action you wanted." Information architecture, a single conversion path, and a crawlable structure are the leverage — polished copy on the wrong pages, in the wrong order, pointing nowhere, still leaks every visitor.

Decide the buyer and the one goal first, then let every page, link, and section serve that goal. Work the depth in [references/ia-and-conversion.md](references/ia-and-conversion.md): page taxonomy and each page's single job, the awareness→consideration→decision path, landing-page message hierarchy, SEO IA, failure modes, red flags, and a worked SaaS example.

## Steps

1. **Define the audiences and the one goal.** Name the two or three buyer types the site serves and the single primary conversion the site exists to win, with secondary conversions ranked beneath it. This step is done when one sentence states the buyer and the one action, and no two actions claim "primary."

2. **Map the page taxonomy.** Draw the page tree (home, product/features, pricing, use-cases, about, blog, docs, contact) and write the single job of each page beside it, per the taxonomy table in the reference. This step is done when every needed job owns one page and no page carries two jobs.

3. **Design the navigation and conversion path.** Set global nav to five-to-seven top-level destinations, place every page on one stage of awareness→consideration→decision, and pick one persistent primary CTA (one verb) for the header and footer. This step is done when the nav holds seven items or fewer and the primary CTA label and destination stay identical across pages.

4. **Order the sections within each landing page.** Sequence the home page and any campaign page top-to-bottom — above-the-fold clarity (outcome headline, who-it-is-for subhead, CTA), then problem/promise, how-it-works, proof, objection handling, closing CTA — following the message hierarchy in the reference. This step is done when clarity precedes proof, proof precedes the closing ask, and one primary action repeats top and bottom.

5. **Wire internal links and SEO structure.** Group content into pillar-and-cluster topics, give every page a flat readable URL, and connect every page to at least one inbound and one outbound internal link. This step is done when no page is an orphan, every URL is a human-readable slug, and each pillar has cluster posts linking up to it.

6. **Review against the conversion path.** Walk the site as the buyer from an awareness entry point to the primary conversion, and check it against the failure modes and red flags in the reference (navigation overload, no clear primary CTA, orphan pages, keyword-stuffing). The review is done when a continuous click path runs entry → consideration → decision and no red flag stands unresolved.
