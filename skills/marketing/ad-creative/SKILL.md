Generate paid-ad creative that earns attention from a scrolling stranger, frames the offer through one sharp lever, and fits the surface it runs on. The angle does the persuading and the hook buys the first three seconds; targeting only decides who gets the chance to scroll past. Lead each concept with one dominant lever so a paid test reads a clean signal.

Work the procedure below. The angle taxonomy, hook patterns, format-fit table, failure modes, red flags, and a worked example live in [references/ad-angles.md](references/ad-angles.md) — read that reference before step 2.

## Steps

1. **Define the audience and the offer.** State the product, the price, the one audience this run targets, the felt pain or desire that audience carries, and the awareness stage (unaware, problem-aware, solution-aware, product-aware, most-aware). This step is done when audience, offer, pain-or-desire, and awareness stage are each written in one line.

2. **Pick three to five distinct angles.** Choose leads from the eight-lever taxonomy (pain, desire, curiosity, social proof, authority, contrarian, us-vs-them, transformation) so the set tests different psychology against the one audience. This step is done when at least three angles are named and each names a different dominant lever.

3. **Write a 3-second hook per angle.** Draft one hook per angle using a named pattern (pattern interrupt, direct call-out, problem-agitate, bold claim, curiosity gap, question, result-first, story cold-open). This step is done when each hook would stop a target stranger on frame one with no sound and no brand name.

4. **Match a format to each angle.** Assign one format (static, carousel, short video / Reels, or UGC) per concept using the format-fit table, sized for Meta or X. This step is done when each concept names one format and that format reinforces its lead lever.

5. **Spec the creative per concept.** Write, per concept, the visual direction, the beat-by-beat script or slide sequence, the on-screen text, and the call-to-action. This step is done when each concept carries a frame-one hook, a beat structure, and one explicit CTA.

6. **Plan the test.** Set the variable held constant (one audience), the variable varied (the angle), the budget split across concepts, and the primary metric (3-second hold rate or click-through) that decides the winner. This step is done when the held variable, the varied variable, and the win metric are each stated.

7. **Screen against the failure modes.** Check the concepts from step 2 against the four failure modes (clever-but-unclear, no hook, single-angle testing, ignoring the platform) plus the red-flag list in the reference. This step is done when each concept clears the four failure modes and no red flag remains open.
